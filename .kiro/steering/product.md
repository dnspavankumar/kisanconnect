# Product Overview

KisanMitra is a multilingual agricultural assistance application designed for farmers in India. The platform provides:

- **Authentication**: Phone-based login with optional farmer ID
- **Dashboard**: Weather information and agricultural insights
- **Chat Interface**: AI-powered agricultural advisory
- **Disease Detection**: Crop disease identification and management
- **News Feed**: Agricultural news and updates
- **Profile Management**: User profile and farm details

## Target Users
Farmers in India, with support for English, Hindi (हिंदी), and Telugu (తెలుగు) languages.

## Key Features
- Multilingual support (en, hi, te)
- Mobile-first responsive design
- Weather integration
- Firebase Authentication with phone OTP
- Firestore database for user profiles
- Mock API for development features
